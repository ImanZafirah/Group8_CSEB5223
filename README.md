# StudySphere: Student Learning Management System (SLMS)

StudySphere is a Student Learning Management System (SLMS) designed to simplify academic administration by centralizing student and course management. It allows administrators to manage student records, courses and enrollments, while enabling students to view available and registered courses. The system ensures that academic information is organized, secure and easily accessible.
